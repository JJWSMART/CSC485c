函数和结构

结构变量：单值变量 = 结构将数据组合成一个单独实体

一个结构可赋给另外一结构

处理基本类型 =》 处理结构 ： 将结构作为参数传递， 在需要时将结构作为返回值使用

缺点： 结构非常大 =〉复制结构将增加内存要求，降低系统运行的速度

变通： 许多C程序员倾向于传递结构的地址，使用指针来访问结构的内容
       C++ 提供了第三种选择 = 按照引用传递

方法1: 结构比较小时，按值传递结构


struct travel_time
{
   int hours;
   int mins;
}

travel_time sum (travel_time t1, travel_time t2);
